
Technical Dependencies:-

Framework: Java Swings

Source code language: Java

OS: Windows

software libraries: MySQL, Eclipse

Steps to run:

1) Open mysql command line,
In library.sql please change the path to csv files and then
Source library.sql file.
-- This creates the library database.

2) Open eclipse - (Eclipse Mars) :
Import the source code. 
Please make sure that you have java swings and window builder in your eclipse.
In Library.java file there is code for sql connection, so please change username and password. (use your mysql username and password)

In the default package java files are present:
-->  MenuOS.java is used for connection to database to get the results.

Run it then a new window appears which is the java application. Then you can find all the required operations in the library management system.
